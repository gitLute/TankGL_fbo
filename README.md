# ТАНКОВАЯ ДУЭЛЬ (TankGL_fbo)

Двухмерная игра для двух игроков на одном экране. Разработана на C# WPF с использованием OpenGL и спрайтовой графики.

## СООТВЕТСТВИЕ ЗАДАНИЮ
- Два игрока управляют танками с одной клавиатуры.
- На поле случайно появляются призовые объекты (ускорение, броня, урон, патроны, топливо).
- Характеристики танка временно изменяются при подборе бонусов.
- Присутствуют стены и система столкновений.
- Победа достигается при уничтожении танка противника.

## ТЕХНОЛОГИИ
| Компонент | Технология |
|---|---|
| Язык и платформа | C# и .NET 10.0 |
| Интерфейс | WPF (основа) + WindowsFormsHost |
| Графика | OpenGL, GLSL шейдеры, спрайты |
| Загрузка изображений | StbImageSharp |
| Тестирование | xUnit |

## АРХИТЕКТУРА И ШАБЛОНЫ
Проект разделен на три части:
- TankGL_fbo.Core: игровая логика, сущности, системы, паттерны.
- TankGL_fbo.WPF: окно WPF, инициализация OpenGL, рендеринг, ввод.
- TankGL_fbo.Core.Tests: модульные тесты ядра.

## Реализованные шаблоны проектирования:
| Шаблон | Файл | Назначение |
|---|---|---|
| Фабричный метод | BonusFactory.cs | Создание бонусов с заданным типом и временем жизни |
| Декоратор | StatDecorator.cs и наследники | Временное изменение характеристик танка без изменения базового класса |

Основные системы:
- SceneManager: переключение между меню, настройками и уровнями.
- InputSystem: обработка клавиатуры и создание снарядов.
- CollisionSystem: проверка столкновений (танки, стены, пули, бонусы).
- SpawnSystem: случайное появление бонусов в свободных местах.

## СБОРКА И ЗАПУСК
Требуется .NET 10.0 SDK ОС Windows, для Linux систем дополнительно потребуется Wine.

Собрать проект можно через следующие скрипты:
| ОС | Команда |
|---|---|
| Windows | ```buildAndRun.bat``` |
| Linux | ```chmod 775 ./buildAndRun.sh ; ./buildAndRun.sh``` |

Ассеты (текстуры, шейдеры, config.json) копируются в выходную папку автоматически.
